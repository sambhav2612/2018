# simple-api

## at v0.0.2

To add a new entry, simply follow this syntax inside [ ]:

```json
{
    "name": "YOUR-FULLNAME",
    "bio": "YOUR-BIO",
    "quote": "YOUR-FAVOURITE-QUOTATION",
    "github_url": [
        "https://github.com/YOUR-GITHUB-USERNAME/"
    ],
    "github_username": "YOUR-GITHUB-USERNAME"
}
```
